## ✨ Features

- ✅ **100% Free** - No signup, no ads, no hidden costs
- ✅ **Live Stock Data** - Real-time NSE/BSE prices
- ✅ **Mobile Optimized** - Works perfectly on all devices
- ✅ **Privacy First** - No data collection, calculations done locally
- ✅ **SEO Optimized** - Fast loading, indexed by Google
- ✅ **Open Source** - MIT licensed

## 🛠️ Built With

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** Shadcn/ui
- **Charts:** Recharts
- **Deployment:** Vercel/Hostinger
